# SearchAndSorted

## This is an object that API for search and sort and paging get :

```json
{
  "entityFilterTermsAndSortParams": {
    "pageIndex": 0,
    "pageSize": 0,
    "entityFilterTermList": [
      {
        "searchTerm": "string",
        "searchValue": "string",
        "searchFilterOp": 0,
        "logicalOp": 0
      }
    ],
    "entitySortParamList": [
      {
        "sortField": "string",
        "sortDir": 0
      }
    ]
  }
}
```
